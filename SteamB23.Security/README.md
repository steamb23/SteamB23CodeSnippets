# SteamB23.Security (C#)
메모리 해킹을 통한 값 수정을 감지하는 기능을 제공하는 프로젝트입니다.

정수 형식을 암호화하는 SecureInt64 구조체와 부동소수점 형식을 암호화하는 SecureDouble 구조체만을 지원하며 이외의 형식에 대한 암호화 구조체는 두 구조체를 통해 사용이 가능하므로 구현되어있지 않습니다.
모든 암호화 형식은 원본 형식처럼 사용할 수 있습니다.
``` c#
SecureInt64 a = 1000;
long b = a + 500;

b == 1500;
```
